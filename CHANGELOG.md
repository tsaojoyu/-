## v1.0 — 2026-05-19

### 新增
- **漁港區塊**完整互動邏輯
  - 四艘船隻色塊熱區（延繩釣船、拖曳網船、有鏢台的船、賞鯨船）
  - Hover 效果：translateY 上移 + 金色邊框光暈 + 船名 tooltip
  - Lightbox：點擊開啟、✕ 關閉、點擊遮罩關閉、ESC 關閉、焦點管理
  - 鍵盤可操作（Tab 選取熱區，Enter / Space 開啟）
- **木刻版畫風色彩系統**（`css/style.css` `:root` 區塊）
- **Banner 與漁場箱庭佔位區塊**（虛線框）
- CSS 色塊場景：天空漸層、山脈輪廓（遠近兩層）、海面水紋、碼頭

### 佔位素材
- 插畫：CSS 色塊與 clip-path 山形，非正式插畫
- 文字：`data/vessels.js` 四艘船說明與訪談節錄標示「佔位，待替換」

### 檔案結構
```
漁港demo/
  index.html          主頁面
  css/
    style.css         全域樣式、色彩變數
    harbor.css        漁港場景 + Lightbox 樣式
  js/
    harbor.js         熱區點擊與 Lightbox 邏輯
  data/
    vessels.js        四艘船隻資料（內容待替換）
  CHANGELOG.md
```

### 已知限制
- 無響應式設計（RWD），以寬螢幕（1200px+）瀏覽為主
- 漁場箱庭互動邏輯尚未實作
- 字型需網路連線載入（Noto Serif TC），離線自動退回系統字型

## v1.1 — 2026-05-19

### 新增
- **Lightbox 媒體區**
  - 影片播放器（`<video>` 原生播放，支援 .mp4；.MOV 需確認瀏覽器相容性）
  - 多張圖片輪播（左右箭頭切換、頁數計數）
  - 影片優先顯示，其次圖片；兩者都沒有則不顯示媒體區
  - 影片關閉 Lightbox 時自動停止
  - .MOV / 不支援格式顯示提示訊息
- `data/vessels.js` 新增 `images`（陣列）和 `video`（路徑或 null）欄位
  - 延繩釣船、有鏢台的船：各配置一支測試影片
  - 所有船隻：配置 picsum.photos 佔位圖片

### 媒體素材更新方式
- **替換照片**：將圖片放入 `images/` 資料夾，在 `vessels.js` 的 `images` 陣列填入路徑（如 `images/longline-1.jpg`）
- **替換影片**：將 .mp4 放入 `images/` 資料夾，在 `vessels.js` 的 `video` 欄位填入路徑
- **新增照片**：在 `images` 陣列新增一筆路徑即可，輪播自動調整
- **移除影片**：將 `video` 設為 `null`

## v2.0 — 2026-05-20

### 架構重構：漁場箱庭互動模式全面改版

#### 互動邏輯
- **拆除單卡片面板**（舊：點擊才顯示、左右翻頁）
- **改為四大常駐橫條**，頁面載入即全部可見，讀者可自行往下捲動
- 插畫熱區改為**錨點導覽**：點擊後平滑捲動到對應橫條，並將目標卡片置中顯示、金色框線短暫閃爍提示
- 各橫條標題列加入「↑ 回到海上」按鈕

#### 類別更名
| 舊類別 | 新類別 |
|--------|--------|
| 山頭・看山版 | 漁場 |
| 魚種 | 魚種（不變）|
| 潮流 | 洋流 |
| 漁民・漁場記憶 | 漁法 |

#### 卡片軌道
- 橫向拖曳滑動，放開後有慣性飄動（速度 × 0.92 衰減）
- **景深效果**：中央卡片放大（scale 1.03）、上浮 10px、深陰影；邊緣卡片縮小（scale 0.91）、貼平、淺陰影；捲動時即時計算
- 目標卡片置中定位：點擊熱區時軌道自動水平捲到對應卡片

#### 視覺個性
- 四個類別各有獨立主色（標題列 + 卡片分隔線 + 引文邊框）
  - 漁場：墨綠（`--mountain-near`）
  - 魚種：海藍（`--sea-surface`）
  - 洋流：深海藍（`--sea-mid`）
  - 漁法：暖棕（`--dock-color`）

#### 佔位資料擴充（Demo 用）
- 各類別由 3 張卡擴充為 5–6 張，足以展示橫向拖曳與景深效果
- 新增佔位項目：美崙山、木瓜溪口、立霧溪口（漁場）；鬼頭刀、黑鮪魚、飛魚（魚種）；黑潮、湧升流（洋流）；延繩釣、鏢魚、定置漁網（漁法）

## v2.2 — 2026-05-20

#### 閱讀模式
- 拖曳慣性滑動 → **翻頁模式**：每次顯示 2 張卡片，以 ← → 按鈕翻頁

- 各橫條拉開 `margin-bottom: 2.5rem`，改為各自獨立邊框與陰影

#### 熱區點擊相容
- 點擊插畫熱區仍可跳到對應橫條，並跳到含目標卡片的分頁 + 金色閃爍
